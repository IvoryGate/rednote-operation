# 小红书推荐算法机制

## 推荐系统架构概述

小红书采用**双列信息流 + 搜索引流**的内容分发架构。推荐算法以用户兴趣为核心，结合内容质量、互动信号、社交关系等多元维度，实现千人千面的个性化推荐。算法核心目标是：将最优质的内容匹配给最感兴趣的用户，最大化用户停留时长与互动深度。

## 双列Feed流机制

### 发现页（推荐Feed）
- **双列瀑布流布局**：每次展示2列，用户主动选择点击感兴趣的内容
- 相比单列信息流，双列要求封面图和标题具有更强的吸引力
- 推荐Feed由算法基于用户兴趣模型动态生成

### 关注页（社交Feed）
- 按时间倒序展示已关注账号的最新发布
- 权重因素：发布时间 > 互动热度
- 关注页流量占比约20-30%，推荐页占比60-70%

### 搜索流量
- **搜索流量占比高达40%以上**，是小红书区别于其他平台的显著特征
- SEO关键词配置是内容运营的核心工作之一
- 搜索结果排序 = 关键词匹配度 × 内容质量分 × 历史互动数据

## 内容冷启动流程

### 第一阶段：小流量测试（发布后0-2小时）
- 新笔记发布后系统分配约200-500次初始曝光
- 根据CTR（点击率）、互动率（点赞/收藏/评论/转发）判断内容质量
- **CTR是冷启动阶段最重要的指标**，封面图直接决定生死
- 此阶段点赞率低于1%或CTR低于5%的内容基本停止推荐

### 第二阶段：扩大推荐（2-24小时）
- 第一阶段数据达标的笔记进入更大流量池（5000-20000曝光）
- 此时算法综合评估：CTR + 完播率 + 互动深度 + 内容质量分
- 互动信号中，**收藏权重最高**（表明内容有长期价值）
- 评论质量比数量重要——有深度的评论比简单表情包更有价值

### 第三阶段：全面推荐（24-72小时）
- 优质笔记进入数十万乃至百万级曝光池
- 算法开始引入更多维度：账号权重、领域相关性、时效性
- 此阶段是内容的"爆发期"，好内容可以在3天内从0到爆款

### 第四阶段：长尾流量（72小时至数月）
- 小红书区别于抖音的核心特征：**优质内容的搜索长尾效应**
- 已发布数月的笔记仍可通过搜索获得稳定流量
- 时效性弱的内容（攻略、教程、测评类）长尾价值最高

## 互动信号权重排序

| 互动行为 | 权重 | 说明 |
|---------|------|------|
| 收藏 | ⭐⭐⭐⭐⭐ | 表明内容有长期保存价值，算法最高权重 |
| 完播率/完整阅读 | ⭐⭐⭐⭐⭐ | 表明内容质量高，用户愿意看完全部 |
| 评论（有内容） | ⭐⭐⭐⭐ | 深度评论促进话题延伸，增加内容权重 |
| 转发/分享 | ⭐⭐⭐⭐ | 社交传播信号，扩展内容影响力 |
| 点赞 | ⭐⭐⭐ | 低成本互动，权重相对较低 |
| 关注作者 | ⭐⭐⭐ | 表明用户对账号认可 |
| 进入作者主页 | ⭐⭐ | 行为信号但权重不高 |
| 评论（表情/无意义） | ⭐ | 低质量互动，权重极低 |

**关键规律**：收藏 > 完播 > 深度评论 > 转发 > 点赞

## 内容质量评分体系

### 图片/封面质量
- 清晰度、构图、色彩搭配
- 封面图与标题的匹配度
- 原创性（非搬运、无水印）
- 美观度与审美水平

### 内容价值
- 信息密度与实用价值
- 结构化程度（清单、步骤、对比等易读格式）
- 真实感（真实体验 > 搬运信息 > AI生成无加工）

### 合规性
- 是否触发审核规则
- 是否为报备的商业内容
- 是否包含违规引流信息

## 账号权重因素

### 基础权重
- 账号注册时长与活跃度
- 内容垂直度（专注单一领域的账号权重更高）
- 历史内容质量表现（过往笔记的平均互动数据）

### 行为权重
- 正常的使用行为（规律发布、正常互动）
- 账号安全等级（无违规记录）
- 创作者等级（社区贡献度积分）

### 降权因素
- 频繁删除/修改笔记
- 批量发布高度相似内容
- 刷量行为（买赞、买粉、互赞群）
- 违规记录（广告未报备、敏感内容）
- 长时间不活跃（30天以上未发布）
- 账号切换设备/IP频繁

## 算法冷知识

1. **新号红利**：没有违规记录的全新账号，冷启动初期反而有轻微流量倾斜
2. **周五晚高峰**：周五18:00-22:00为平台流量最高峰，发布竞争也最激烈
3. **非高峰发布策略**：工作日10:00-12:00或周末早上发布，竞争较小，冷启动更友好
4. **标签算法**：系统通过图片识别和文本NLP自动打标签，与作者主动打的标签共同决定推荐分发
5. **本地化权重**：带位置标签的内容在对应城市用户中有额外曝光权重
6. **重复点击惩罚**：同一笔记被同一用户反复点击不互动，反而可能降低该笔记的推荐权重
