# 策略制定提示词

> 用于AI运营智能体进行小红书内容策略、增长策略和运营规划的LLM提示词。

---

## 1. 内容日历规划

### 月度内容计划

```
请为一个小红书[类目]账号制定下一个月的内容日历。

账号信息：
- 账号定位：[定位描述]
- 粉丝量级：[粉丝数]
- 当前类目：[类目]
- 核心目标：[增粉/变现/品牌建设]

输出格式：

【第1周：MM/DD - MM/DD】
| 日期 | 内容主题 | 内容形式 | 核心关键词 | 发布时段 |
|------|----------|----------|------------|----------|
| 周一 | ... | 图文 | ... | 12:00 |
| 周三 | ... | 视频 | ... | 20:00 |
| 周五 | ... | 图文 | ... | 18:30 |

（共4周）

要求：
1. 每周至少1篇干货教程型内容（涨粉）
2. 每周至少1篇互动型内容（提活跃）
3. 标记出适合投流的潜力内容
4. 考虑当月热点节日和话题
5. 内容之间有连贯性，形成系列感
6. 不同类型内容穿插发布，避免单调

附加：
- 每周重点话题标签推荐（5个）
- 封面风格建议
- 可预埋的互动话题
```

### 周度执行方案

```
请根据以下月度内容计划，制定下周的详细执行方案：

月度计划：[粘贴月度计划]
本周重点：[本周目标]
当前账号状态：[数据情况]

输出：
1. 每日发布计划（具体到时间段）
2. 每条笔记的标题方向和封面思路
3. 内容素材准备清单
4. 互动维护计划（评论回复、私信跟进）
5. 数据追踪指标设置
6. 应急预案（内容储备/备选方案）
```

## 2. 话题标签策略优化

### 账号标签矩阵设计

```
请为以下小红书账号设计标签矩阵策略：

账号信息：
- 类目：[类目]
- 内容方向：[方向]
- 目标受众：[描述]

设计要求：

1. 三层标签体系
   - 品牌标签（3-5个）：账号专属，每篇必带
   - 类目标签（5-8个）：行业大类，持续覆盖
   - 流量标签（10-15个）：根据内容灵活更换

2. 标签组合策略
   - 每篇笔记的标签组合规则（大+中+小）
   - 不同内容类型的标签侧重
   - 蹭热点标签的安全边界

3. 标签优化机制
   - 标签效果评估周期
   - 淘汰低效标签的标准
   - 新标签测试方法

输出：完整的标签矩阵表格+使用说明。
```

### 单篇标签优化

```
请为以下小红书笔记推荐最优标签组合：

笔记主题：[主题]
内容类型：[教程/测评/分享/其他]
目标人群：[描述]
内容关键词：[关键词]

约束条件：
- 标签总数：5-10个
- 必须包含账号品牌标签：[品牌标签]

请给出3套不同策略的标签方案：

方案A - 精准覆盖型（适合涨粉）
方案B - 流量最大化型（适合曝光）
方案C - 转化导向型（适合变现）

每套方案标注：
- 每个标签的预估搜索热度
- 竞争度评估
- 适用场景说明
```

## 3. 发布排期建议

### 最佳发布时间分析

```
请根据以下账号数据，推荐最优发布排期：

账号类目：[类目]
目标受众画像：[受众描述]
历史发布数据：[可选：粘贴30天发布时间与互动数据]

分析内容：
1. 推荐发布时间段（精确到小时）
   - 周一到周五的最佳发布时间
   - 周末的最佳发布时间
   - 不同内容类型的发布时间建议

2. 发布频率建议
   - 当前粉丝阶段的最佳发布频率
   - 不同内容形式的配比建议
   - 特殊时期（大促/节日）的发布节奏

3. 发布队列策略
   - 优质内容放在高流量时段
   - 互动型内容放在低谷时段
   - 测试新内容方向的最佳时机

4. 注意事项
   - 避免的发布时间段
   - 同一天多条内容的间隔建议
   - 更新时间稳定性对账号权重的影响
```

### 投流策略建议

```
请为以下小红书笔记制定投流（薯条/聚光）策略：

笔记信息：
- 内容主题：[主题]
- 目标：[增粉/互动/转化]
- 预算：[每日预算]

投流方案：

1. 自然流量验证期（发布后2-4小时）
   - 判断是否值得投流的数据标准
   - 自然互动率达到多少开始投流

2. 投流执行方案
   - 推荐投放时间
   - 定向设置建议（性别/年龄/城市）
   - 预算分配策略

3. 效果评估
   - 关键监控指标
   - 停止投放的信号
   - 复盘评估框架
```

## 4. 增长策略

### 从0到1冷启动

```
请为一个小红书新账号制定从0到1的冷启动增长策略：

账号信息：
- 类目：[类目]
- 内容定位：[描述]
- 团队资源：[单人/小团队]
- 目标：30天涨粉至[目标粉丝数]

策略框架：

1. 账号装修期（第1-3天）
   - 账号命名、头像、简介策略
   - 主页视觉风格搭建
   - 前3篇内容的定位策略

2. 内容启动期（第4-14天）
   - 每天1更的内容节奏
   - 爆款公式内容（高收藏+高搜索）
   - 第一批种子用户的获取

3. 增长加速期（第15-30天）
   - 系列化内容建立专业度
   - 互动运营提升账号权重
   - 合作互推/评论区截流技巧

4. 关键指标
   - 各阶段的核心KPI
   - 需要调整策略的危险信号
   - 可继续放大的成功信号

每个阶段给出具体可执行的操作清单。
```

### 粉丝增长突破

```
当前账号已进入增长瓶颈期，请制定突破策略：

账号现状：
- 当前粉丝：[粉丝数]
- 近30天平均增粉：[数字]
- 最近3个月粉丝增长趋势：[上升/持平/下降]
- 主要问题：[内容同质化/互动下降/流量瓶颈]

增长突破方案：

1. 内容升级策略
   - 哪些内容方向有扩圈潜力
   - 内容形式是否需要调整
   - 系列IP的打造方案

2. 流量渠道拓展
   - 搜索SEO优化方案
   - 跨平台引流策略
   - KOL/KOC合作互推

3. 用户运营策略
   - 粉丝互动率提升方案
   - 社群/私域引流
   - 用户生成内容（UGC）激励

4. 资源投入建议
   - 投流预算分配建议
   - 时间/精力投入优先级
   - 外部资源获取路径

输出3个月内可实现的具体增长目标。
```

### 变现路径规划

```
请为以下小红书账号制定变现策略：

账号信息：
- 类目：[类目]
- 粉丝量级：[粉丝数]
- 当前变现方式：[品牌合作/带货/知识付费/无]
- 目标月收入：[目标金额]

变现方案：

1. 变现定位
   - 适合的变现模式组合
   - 各变现模式的优先级排序
   - 启动时间节点规划

2. 品牌合作策略
   - 合适的品牌画像和报价策略
   - 商单内容与日常内容的配比（建议<=20%）
   - 合作案例集和数据包的准备

3. 带货策略（如适用）
   - 选品标准和供应链建议
   - 带货内容形式设计
   - 直播带货的启动计划

4. 知识付费策略（如适用）
   - 可付费化的内容资产
   - 产品设计（课程/社群/咨询）
   - 发售节奏和预热方案

5. 收入预期
   - 分阶段的收入目标（3/6/12个月）
   - 各变现渠道的预期占比
   - 成本预算和ROI预估
```
