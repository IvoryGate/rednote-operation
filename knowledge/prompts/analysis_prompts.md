# 数据分析提示词

> 用于AI运营智能体进行小红书数据分析和账号诊断的LLM提示词。

---

## 1. 竞品账号分析

### 基础竞品分析

```
请对以下小红书账号进行竞品分析：

竞品账号：[账号名称/ID]
所属类目：[类目]
分析周期：近30天

请从以下维度分析：

1. 账号基础信息
   - 粉丝量级、笔记总数、获赞与收藏总数
   - 账号定位与内容矩阵

2. 内容表现分析
   - 平均互动率（点赞+收藏+评论+分享）/曝光量
   - 爆款率（互动>1000的笔记占比）
   - 内容形式偏好（图文vs视频占比）

3. 内容策略
   - 高频关键词和话题标签
   - 封面风格和视觉统一性
   - 更新频率和发布时间规律

4. 变现分析
   - 商单内容占比估算
   - 合作品牌类型和数量
   - 带货内容表现

5. 优劣势总结
   - 账号的核心竞争力
   - 内容短板和可改进空间
   - 可借鉴的策略点

输出格式：结构化报告，每个维度用表格或列表呈现。
```

### 深度对比分析

```
请对比分析以下两个小红书账号，评估竞争态势：

账号A：[账号名称]
账号B：[账号名称]

对比维度：
1. 粉丝增长趋势（近90天）
2. 互动率对比（按内容类型分类）
3. 内容质量评分（封面、标题、正文、标签）
4. 用户画像重合度
5. 变现路径差异

最终输出SWOT对比矩阵：

| 维度 | 账号A | 账号B |
|------|-------|-------|
| 优势 | ...   | ...   |
| 劣势 | ...   | ...   |
| 机会 | ...   | ...   |
| 威胁 | ...   | ...   |

给出差异化竞争建议。
```

## 2. 内容表现分析

### 单篇笔记诊断

```
请对以下小红书笔记进行表现分析：

笔记链接/ID：[ID]
发布时间：[日期]
当前数据：
- 曝光量：[数字]
- 阅读量：[数字]
- 点赞：[数字]
- 收藏：[数字]
- 评论：[数字]
- 分享：[数字]

分析维度：
1. 健康度评估
   - 点击率（阅读/曝光）是否达标（基准15%+）
   - 互动率是否达标（基准5%+）
   - 收藏/点赞比是否合理（干货类应>0.5）

2. 内容诊断
   - 标题是否包含搜索关键词
   - 封面是否具备吸引力
   - 首段是否有钩子
   - 正文信息密度是否足够
   - 标签策略是否合理

3. 改进建议
   - 针对低点击率：改进封面和标题
   - 针对低互动率：优化内容价值和引导
   - 针对低收藏率：增加干货密度

输出改进方案，最多3条最关键建议。
```

### 批量内容复盘

```
以下是我近30天发布的小红书笔记数据（CSV格式）：

[粘贴数据]

请帮我分析：

1. 整体表现概览
   - 总曝光、平均互动率、爆款率
   - 最佳和最差表现笔记

2. 表现规律发现
   - 哪种内容类型表现最好？
   - 哪个发布时间段效果最优？
   - 哪些关键词带来了更多流量？
   - 图文vs视频的表现差异

3. 优化方向
   - 找到可复制的爆款模式
   - 识别低效内容类型（建议减少或改进）
   - 每周内容配比建议

输出结论需有数据支撑，避免主观判断。
```

## 3. 热点话题识别

### 行业热点发现

```
请分析[类目]类目在小红书近期的内容趋势：

分析范围：近30天
数据来源：根据以下样本内容推断
[可选：粘贴热门笔记标题/关键词]

请回答：
1. 当前TOP10热门话题标签是什么？
2. 哪些内容方向最近流量增长最快？
3. 有哪些处于上升期的新兴话题？（竞争低、增长快）
4. 当前用户搜索量上升的关键词有哪些？
5. 头部博主在讨论什么话题？

输出格式：
- 热度趋势列表（高热度/增长中/衰退中）
- 每个话题的流量天花板预估
- 切入建议（如何差异化）
```

### 蓝海话题挖掘

```
请基于以下类目和关键词，挖掘小红书上的蓝海话题机会：

类目：[类目]
核心关键词：[关键词]

分析维度：
1. 搜索量分析
   - 与核心关键词相关的长尾搜索词
   - 月搜索量估算（高/中/低）
   - 竞争笔记数量估算

2. 竞争分析
   - 现有相关内容的质量水平
   - 头部笔记的互动情况
   - 内容缺口识别（用户想看的但目前内容少）

3. 蓝海判断
   - 高搜索低竞争的话题优先推荐
   - 从其他平台（抖音/知乎/B站）迁移的热门话题
   - 季节性潜在热点

推荐TOP5蓝海话题，每个附上内容切入建议。
```

## 4. 受众洞察提取

### 评论区分析

```
请对以下小红书笔记的评论区进行受众洞察分析：

笔记主题：[主题]
评论样本（最近50条）：[粘贴评论]

分析目标：
1. 用户关注点提取
   - 最高频提及的问题/疑虑
   - 用户最在意的产品/内容维度
   - 未被满足的需求暗示

2. 用户画像推断
   - 从评论推断受众年龄段、消费力、使用场景
   - 用户身份特征（学生/职场/宝妈等）

3. 情绪分析
   - 正面/负面/中性评论比例
   - 高频情绪词提取

4. 内容优化建议
   - 基于评论反馈的改进方向
   - 可做续篇/衍生内容的话题点
   - 评论区高频词可作为标签/标题参考
```

### 粉丝画像构建

```
请基于以下账号数据，构建该账号的粉丝画像：

账号名称：[名称]
类目：[类目]
近30天数据：[粘贴数据]

输出粉丝画像报告：
1. 人口特征
   - 年龄分布（主年龄段及占比）
   - 性别比例
   - 城市线级分布
   - 消费力等级

2. 兴趣特征
   - TOP5兴趣类目
   - 内容消费偏好（图文vs视频vs直播）
   - 活跃时间段

3. 消费行为
   - 价格敏感度
   - 购买决策影响因素
   - 信任建立方式

4. 运营建议
   - 内容策略调整建议
   - 变现方式匹配建议
   - 粉丝运营（私域/社群/直播）建议
```
